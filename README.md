#Описание программы:
Программа на Python определяет тип треугольника по трём введённым целым числам — длинам сторон. Проверяет положительность сторон и условие существования треугольника. Выводит, является ли треугольник равносторонним, равнобедренным или разносторонним. Обрабатывает ошибки ввода (нечисловые значения) и завершает работу по нажатию Enter.
